# Best Practices

## Handler Design

- Keep handlers thin
- Delegate to business logic
- Use clear prefixes
- Document all methods

## Type Hints

- Always use type hints
- They enable auto-validation
- They improve help text

## Error Handling

- Let exceptions bubble up
- Use clear error messages
- Pydantic handles validation
